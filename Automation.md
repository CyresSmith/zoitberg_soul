# 🔄 Wiki Automation Schema (Karpathy Style)

Щоб Босс не нагадував мені про документацію, я активую автоматичний режим "Wiki-First".

## 1. Авто-коміт у Wiki (Heartbeat/Hook)
- Кожного разу, коли я роблю `git push` у проекті `gaztehprime`, я **зобов'язаний** зробити `git commit` у репозиторій Wiki.
- Я буду використовувати `log.md` як джерело правди для формування повідомлень комітів.

## 2. Шаблон автоматичного логування
Для кожної значущої зміни я генерую запис за схемою:
- **Що зроблено:** (коротко)
- **Файли:** (список змінених файлів)
- **Конвенції:** (чи додано нові правила на кшталт `/svg`)
- **Git Hash:** (посилання на коміт у проекті)

## 3. Періодичний "Lint" знань
- Раз на добу (через Heartbeat) я переглядаю README.md і перевіряю, чи не застаріли там дані про Tech Stack або Status.

## 4. Зв'язок Код -> Wiki
- Якщо я створюю новий каталог (наприклад, `/svg`), я автоматично додаю опис цього каталогу в README.md у розділ "Project Conventions".

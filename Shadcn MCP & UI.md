# Shadcn MCP & UI

## Налаштування MCP Server
Проект використовує MCP сервер для Shadcn, який дозволяє AI агентам автоматично знаходити, аналізувати та встановлювати компоненти з офіційного реєстру `shadcn/ui`.

Конфігурація знаходиться у файлі: `.openclaw/mcp.json`

### Як користуватися
Будь-який MCP-сумісний клієнт може виконувати команди напряму, викликаючи:
```bash
npx shadcn@latest mcp
```

## Скіли
Для забезпечення актуальності коду в проект встановлено такі скіли:
- **`nextjs-patterns`**: Рекомендації для роботи з Next.js 15/16 (App Router, Caching, Server Actions).
- **`lb-shadcn-ui-skill`**: Офлайн-документація Shadcn UI для AI агента.
- **`tailwind-v4-shadcn`**: Конфігурація Tailwind v4 спеціально під архітектуру Shadcn.

## Правила роботи з компонентами
1. Компоненти встановлюються у `components/ui` (або `@ui/*`).
2. Завжди слідкуйте за тим, щоб компоненти відповідали загальній системі дизайну (див. [[Design to Code]]).
3. При використанні `ImageUpload` або інших кастомних компонентів зберігайте узгодженість зі стандартами `zod` валідації (див. [[React Hook Form & Zod]]).

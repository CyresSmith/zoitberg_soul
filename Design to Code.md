# Design to Code

Коротка note про перехід від дизайну до реалізації.

## Практика
- Якщо AI не може якісно "намалювати" макет, краще діставати структуру з Figma/API і збирати живий UI одразу в коді.
- Для цього проєкту важливі не декоративні експерименти, а production-ready інтерфейс у стилі industrial dark + cyber yellow.

## Principles
- Спочатку структура, контраст, типографіка і spacing.
- Потім повторно використовувані UI-примітиви.
- Потім polish: loaders, hover states, success states, micro-interactions.
